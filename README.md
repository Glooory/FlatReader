### FlatReader

一款简单的资讯阅读应用，内容包括知乎日报、干货集中营(仅 Android 方面)、IT之家。  

是自己的一个简单的练手项目，基于 MVP + RxJava + Retrofit + Glide + greenDAO

#### 主要应用到的知识点

- 采用了 MVP 架构
- RxJava + Retrofit 进行网络请求，配置 OkHttp 缓存机制
- 使用 greenDAO 进行已读记录的增、删、改、查
- 使用 RxBus 进行组件间通信
